tmux Basics


    $ tmux new        # neue tmux Session starten
Taste Aktion
Strg b + % vertikal teilen
Strg b + " horizontal teilen
Strg b + zum Fenster wechseln

tmux ist ein Terminal-Multiplexer, der es ermöglicht, innerhalb eines einzigen Terminal-Fensters mehrere unabhängige und dauerhafte Sitzungen zu erstellen und zu steuern.
